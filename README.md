# 🔐 User Access Automation using GitHub Actions

Automate temporary user creation in Linux servers with expiry-based access control.

## 🚀 Features
- User creation with expiry
- Secure password generation
- Automated via GitHub Actions

## ▶️ Run
Go to Actions → Run Workflow

Inputs:
- username
- expiry_date (YYYY-MM-DD)
